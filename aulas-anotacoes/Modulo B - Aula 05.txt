'=' significa Recebe

*No JavaScvript moderno, além de utilizar a palavra 'var', também podemos usar a palavra 'let'*

Existem 3 formas de delimitar uma String no JavaScript

Nome de uma variável(chamados Indicadores)
* Podem começar com 'letra', '$' OU '_'
* Não podem começar com 'numeros'
* É possivel usar 'letras' ou 'numeros'
* É possivel usar 'acentos' e 'simbolos'
* Não podem conter 'espaços'
* Não podem ser 'palavras reservadas' (ex. var)

Dicas para nomes de Indicadores(variáveis)
* 'Miúsculas' e 'minúsculas' fazem diferença
* Tente escolher 'nomes coerentes' para variáveis
* Evite se tornar um 'programador alfabeto' ou um 'programador contador'
